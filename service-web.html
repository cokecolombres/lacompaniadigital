Tipos de Web

Sitio web estático
Se tratan de las páginas webs que solo muestran información permanente tratando de crear un atractivo y enseñando otra salida, ya sea a redes sociales o dar a conocer comercios.
Estas webs no tienen ningún tipo de base de datos, aplicativos especiales.
Sitio web dinámico
Son las páginas más usadas de la actualidad con funciones avanzadas, importantes e incluso cierre de ventas con sistemas totalmente automatizados.
Básicamente puedes crear una herramienta por la cual las personas visiten tu página web o puedes tener incluso una tienda que se retroalimenta.
E commerce
Básicamente se trata de crear una página web que incluye una tienda donde puedas ver el producto, conocer las tallas, saber la disponibilidad, conocer hacia qué país lo envían e incluso pagar directamente en la plataforma. Lo que hace que los procesos de venta se optimicen y se garantice perfectamente la compra de un producto o servicio.
